# Parental Mediation Parent Survey

This is a bilingual survey (English and Chinese) for parents to assess their parental mediation strategies regarding children's internet use.

## Features

- Bilingual interface (English and Chinese)
- All survey items on a single page
- Automatic score calculation for subscales
- Visual insights with bar charts and radar charts
- Responsive design

## Survey Sections

1. **Parental Mediation Strategies** - 5 subscales:
   - Active Mediation
   - Co-use
   - Restrictive Mediation
   - Diversion Mediation
   - Health Mediation

2. **Technical Mediation** - Use of technical tools

3. **Parental Monitoring** - Monitoring behaviors

## Usage

Open `index.html` in a web browser to start the survey. Fill out all questions and click "Calculate Scores" to see your results with charts.

## Scoring

Scores are calculated as averages for each subscale, providing insights into different aspects of parental mediation.
